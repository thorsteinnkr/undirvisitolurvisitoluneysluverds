Birting á breytingu á undirliðum vísitölu neysluverðs á ársgrundvelli.

Rauðu flöggin fást fyrir hverja 2,5% hækkun á milli ára.

Gulu línurnar fást fyrir meira en 7,5% hækkun á milli ára.
